## 🤖 PROJETO EM DESTAQUE: MCP LAB AGENT

**Sistema de QA Autônomo com IA** — projeto open-source que criei e mantenho  
**GitHub:** [github.com/Wesley-Gomes93/mcp-lab-agent](https://github.com/Wesley-Gomes93/mcp-lab-agent) | **Demo:** [qa-lab-sepia.vercel.app](https://qa-lab-sepia.vercel.app) | **npm:** `mcp-lab-agent`

Quando um teste falha, retorna **causa e correção em ~30 segundos** — sem cavar em stack traces. Testes que se autocorrigem e aprendem a cada fix.

| Destaque | Impacto |
|----------|---------|
| Diagnóstico em linguagem clara | "Por que falhou?" em segundos, não em horas |
| Testes autocorretivos com learning system | Taxa de sucesso melhora a cada correção aplicada |
| Integração IDE (Cursor) + Slack Bot | QA no fluxo da conversa |
| Suporte a 11+ frameworks | Cypress, Playwright, WebdriverIO, Appium, Jest, Detox, Robot Framework |
| Detecção automática de flaky | Padrões: timing, selector, element_not_visible |
| Enterprise-ready | Learning Hub, CI/CD, Ollama (offline) |

**Stack:** TypeScript, Node.js, MCP (Model Context Protocol), Groq/Gemini/OpenAI, Vitest, GitHub Actions  
*Publicado no npm | Licença MIT*
