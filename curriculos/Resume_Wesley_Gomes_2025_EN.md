## 🤖 FEATURED PROJECT: MCP LAB AGENT

**AI-Powered Autonomous QA System** — open-source project created and maintained by me  
**GitHub:** [github.com/Wesley-Gomes93/mcp-lab-agent](https://github.com/Wesley-Gomes93/mcp-lab-agent) | **Demo:** [qa-lab-sepia.vercel.app](https://qa-lab-sepia.vercel.app) | **npm:** `mcp-lab-agent`

When a test fails, returns **cause and fix in ~30 seconds** — no digging through stack traces. Self-healing tests that learn from each fix.

| Highlight | Impact |
|-----------|--------|
| Plain-language diagnosis | "Why did it fail?" in seconds, not hours |
| Self-healing tests with learning system | Success rate improves with each applied fix |
| IDE (Cursor) + Slack Bot integration | QA in the conversation flow |
| Support for 11+ frameworks | Cypress, Playwright, WebdriverIO, Appium, Jest, Detox, Robot Framework |
| Automatic flaky detection | Timing, selector, element_not_visible patterns |
| Enterprise-ready | Learning Hub, CI/CD, Ollama (offline) |

**Stack:** TypeScript, Node.js, MCP (Model Context Protocol), Groq/Gemini/OpenAI, Vitest, GitHub Actions  
*Published on npm | MIT License*
